# Core Hook

> Unified settings hub for rimir plugins in the ControlPanel

## Key features

* **Dynamic discovery** -- automatically finds all installed `$:/plugins/rimir/*` plugins
* **Vertical sidebar** -- plugin list with icons, names, and version badges
* **Auto-discovery of settings** -- transcludes each plugin's `settings` tiddler as content panel
* **Fallback panel** -- plugins without settings show an info card with link to documentation

## Prerequisites

No external prerequisites.

## Quick start

Install plugin. All rimir plugins with a `settings` tiddler automatically appear in ControlPanel > Settings > Rimi Plugins.

## Plugin Library

Install from the [rimir plugin library](https://rimir-cc.github.io/tw-plugin-library/) via *Control Panel → Plugins → Get more plugins*.

## Demo
